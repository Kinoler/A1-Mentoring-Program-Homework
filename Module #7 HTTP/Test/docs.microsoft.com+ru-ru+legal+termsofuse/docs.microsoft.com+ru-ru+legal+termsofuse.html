docs.microsoft.com - Условия использования

Принятие Условий

Следующие Условия использования («УИ») применяются к использованию вами веб-сайта Microsoft Docs (https://docs.microsoft.com), Технического профиля (Tech Profile) (https://techprofile.microsoft.com) и любых связанных с ними служб. Microsoft оставляет за собой право обновлять УИ в любое время без предварительного уведомления. Последнюю версию УИ можно просмотреть по ссылке «Условия использования», которая находится внизу наших веб-страниц.

Описание службы

Посредством веб-сайта Microsoft Docs, Технического профиля и любых связанных с ними служб Microsoft предоставляет доступ к разнообразным ресурсам, включая интерактивные учебные пособия, документацию, видео, средства для разработчиков, области загрузки, форумы и сведения о продуктах (совместно именуемые «Службами»). На Службы, включая любые обновления, усовершенствования, новые функции и (или) добавление любых новых веб-свойств, распространяется действие УИ.

Использование только в личных и некоммерческих целях.

Если не указано иное, Службы предназначены для использования в личных и некоммерческих целях. Без предварительного письменного согласия Microsoft Вы не имеете права модифицировать, копировать, распространять, пересылать, отображать, публично демонстрировать, воспроизводить, публиковать, лицензировать, передавать и продавать какую-либо информацию, программное обеспечение, продукты или услуги, полученные через данные Службы, а также создавать на их основе производные работы (за исключением тех случаев, когда это предназначается для вашего собственного, личного, некоммерческого использования).
